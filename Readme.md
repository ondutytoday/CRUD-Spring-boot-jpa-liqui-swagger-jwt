# CRUD REST CONTROLLER

* Это копия приватного репозитория на bitbucket.
* В данном проекте использован следующий стек технологий, фреймворков и библиотек:
    - Java 8
    - Maven
    - Lombok
    - Mapstruct
    - Swagger UI
    - Spring Boot
    - Spring Web
    - Spring Security
    - JPA/Hibernate 
    - PosgreSQL
